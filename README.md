# dotfiles

## Requirements

- Install zsh and [oh-my-zsh](https://ohmyz.sh/#install)
- Install [nix](https://nixos.org/download)

## Installing

Clone repository

```sh
git clone https://github.com/csvenke/dotfiles.git ~/.dotfiles
```

Run install script

```sh
sh ~/.dotfiles/scripts/install.sh
```

## Removing

Run clean script

```sh
sh ~/.dotfiles/scripts/clean.sh
```

Remove repository

```sh
rm -rf .dotfiles
```
